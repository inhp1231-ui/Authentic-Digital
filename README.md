# 🏛️ Authentic Digital - Architecture of Success

A premium digital library website offering curated e-books and resources for digital growth, earning strategies, and style inspiration.

🌐 Live Website: [https://authentic-digital.netlify.app](https://authentic-digital.netlify.app)

---

## 📚 About The Project

Authentic Digital is a responsive, modern web platform designed to showcase digital products across three main categories:

- Authentic Collection - Premium digital assets and resources
- Earning Hack - Strategies and guides for digital income
- Fashion/Style - Style inspiration and digital fashion guides

The website features a clean, professional design with a focus on user experience and mobile responsiveness.

---

## 🎯 Features

- ✅ Fully responsive design (works on mobile, tablet, and desktop)
- ✅ Dedicated category pages for each product line
- ✅ Modern, gradient-based UI with blueprint aesthetic
- ✅ Smooth navigation with dedicated navbar
- ✅ Optimized for performance on all devices
- ✅ 3D elements disabled on mobile for smooth scrolling

---

## 🛠️ Built With

- HTML5 - Structure
- CSS3 - Styling & responsive design
- JavaScript - Interactivity
- AntiGravity - AI-powered development
- GitHub - Version control
- Netlify - Hosting & deployment
